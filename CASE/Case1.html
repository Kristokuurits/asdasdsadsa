Upper Case tool

„ülemise taseme" CASE-vahendid (upper CASE tools) toetavad analüüsi ja projekteerimist.
Peamiselt on nad kasutusel kasutajanõuete analüüsimisel ja dokumenteerimisel.
Nad on ennekõike mõeldud visualiseerimiseks, erinevate skeemide koostamiseks ja ka dokumentatsiooni genereerimiseks.
nad toetavad traditsiooniliste diagrammikeelte kasutamist (olem-seos diagrammid, andmemudelid, UML-skeemid, jne).

Rational Rose: Rational Rose XDE, tarkvaraarendajatele mõeldud laiendatud arenduskeskkond, integreerub Microsoft Visual Studio .NET ja Rational Application Developeriga. Selle tarkvara kirjutas varem Rational Rose’i tootnud IBMi Rational Software osakond.
